# LocalSDR
Autor: Roger Cruz Roura

Context: Simular la instal·lació d’una antena SDR (software defined radio) a l’institut per captar i analitzar senyals de ràdio a través de un programa per Internet i permet escoltar diferents freqüències.

Abast: Entendre el funcionament de la tecnologia SDR. Dissenyar teòricament la instal·lació d’una antena SDR al centre. Identificar el material i el programari necessaris per al sistema. Comparar els avantatges del SDR respecte a la ràdio convencional.
